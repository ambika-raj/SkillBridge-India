# 🇮🇳 SkillBridge India

SkillBridge India is a full-stack web application designed to bridge the gap between education and employment by providing skill development programs, certifications, and job opportunities on a single platform.

The platform supports multiple user roles including Students, Instructors, Employers, and Admins, where each user gets a dedicated dashboard with role-specific functionalities.

# 📌 Features

## 🔐 Authentication System
- User Registration & Login
- JWT-based Authentication
- Role-Based Access Control (RBAC)
- Secure Password Hashing using bcrypt
- Protected Routes

---

## 📚 Courses & Skills Module
- Create, Read, Update, Delete (CRUD) Courses
- Course Categories
- Enrollment System
- Progress Tracking
- Course Management

---

## 🏆 Certification Module
- Auto-generated Certificates
- Unique Certificate Verification ID
- Certificate Tracking

---

## 💼 Jobs Module
- Employers can post jobs
- Students can apply for jobs
- Resume Upload Feature
- Application Status Tracking
  - Pending
  - Reviewed
  - Accepted
  - Rejected

---

## 📊 Role-Based Dashboards

### 👨‍🎓 Student Dashboard
- View enrolled courses
- Track progress
- View certifications
- Apply for jobs

### 👨‍🏫 Instructor Dashboard
- Create and manage courses
- View student enrollments
- Monitor analytics

### 🏢 Employer Dashboard
- Post jobs
- Manage applicants
- Track recruitment process

### 👨‍💼 Admin Dashboard
- Manage users
- Monitor courses and jobs
- Platform analytics
- System activity tracking
